# Strings

> Strings files are used to get language specific display names, descriptions, etc.
>
> Preferred layout for localized content:
>
> - `scripts/content/Strings/en/<file>.json`
> - `scripts/content/Strings/zh-CN/<file>.json`
> - `scripts/content/Strings/ja/<file>.json`
>
> Backward compatibility is preserved for legacy files directly in `scripts/content/Strings`.

## Files

- `Strings/1_6_Strings`
- `Strings/BigCraftables`
- `Strings/Locations`
- `Strings/NPCNames`
- `Strings/Objects`
- `Strings/StringsFromCSFiles`
